# NEDEN PARA İŞLEMLERİNDE DECIMAL KULLANILIR?

# --------------------------------------------------

# 

# double veri tipi, ikili sayı sistemi nedeniyle küçük hassasiyet hatalarına

# yol açabilir.

# 

# decimal veri tipi ise finansal hesaplamalar için tasarlanmıştır ve

# para, maaş, fiyat gibi işlemlerde daha doğru sonuçlar üretir.

# 

# Bu nedenle finansal işlemlerde decimal kullanımı tercih edilir.
